On this site you will get a brief introduction into blender

What even is Blender?

  • Blender is an open source 3d modelling Program

  • It was programmed using C,C++ and Python

  • Blender's vision is that anyone can get a decent 3d modeling program without spending a lot of money

How do I use the General Viewport

  • By pressing the mouse wheel you rotate around the center of the viewport

  • Use Shift and press the mouse wheel to move in the viewport

How is Blender structured?

The red line is the X axis. The green line is the Y axis. There is also a blue line on the Z axis. It runs from bottom to top.

In Blender one often speaks of verticies, edges and also faces Faces are rough surfaces Verticies are small dots at corners of faces Edges are lines on the side of the faces (edges)

What do these parameters mean?

<axis> This is an optional parameter that specifies the axis. These are x, y and z. If released, it performs the action on each axis.

<number>: This is an optional parameter that specifies degrees for rotations, for example. The number can also be determined by mouse movements.

The most important keybinds!

The simplest but also the most important key combinations are:

S <number> (changes the size of an object)

R <axis> <number> (rotates an object)

G <axis> <number> (moves an object)

TAB (switches to edit mode)

E <axis> <number> (Extrudes an object. Also called expand)

Z (Opens a menu for changing the shading)

Shift + A (Opens the Add Menu and lets the user select one of the simple default items)

Edit Mode

This mode is where you spend most of your time!

Any of the previous key combinations also work in this mode

In this mode, there are three icons on the top left, they are Verticie Select, Edge Select and Face Select.
